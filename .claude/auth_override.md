# Smart-Ops-Term 项目授权覆盖

## 🎯 当前授权级别: Level 1 (Auto - 完全自动模式) ⚡

### 📋 项目说明

Smart-Ops-Term 是一个 AI 辅助远程运维终端工具，使用 Python + PyQt6 开发。

**项目特点**:
- 开发中的项目（v1.6.0）
- 快速迭代，频繁修改代码
- 注重功能开发和调试
- 实验性功能较多

### 🔧 授权规则

#### Level 1 (Auto) - 自动执行 ✅ **当前级别**
- 读取项目文件
- 搜索代码（Grep, Glob）
- 解释代码逻辑
- 查看项目结构
- 运行测试脚本
- **编写/编辑源代码文件** (`src/**/*.py`)
- **重构代码**
- **添加新功能**
- **修复 Bug**
- **修改配置文件** (`.env`, `requirements.txt`)
- **创建/编辑文档** (`*.md`)
- **格式化代码**
- **删除源代码文件**
- **Git 提交和推送**
- **安装/卸载依赖包** (`pip install`, `pip uninstall`)
- **运行应用** (`python src/main.py`)
- 修改核心架构文件
- 删除 `.pyc` 缓存文件

#### Level 2 (Notify) - 显示后执行
- (已降级到 Level 1)

#### Level 3 (Confirm) - 需要确认
- (已降级到 Level 1)

#### Level 4 (Approve) - 严格批准
- **删除多个文件**
- **修改数据库 schema**
- **清理缓存/构建产物**
- 变更 API 接口
- 修改 `.git` 目录

#### Level 5 (Forbidden) - 禁止执行
- Git force push
- 删除 `.git` 目录
- 访问敏感文件（密码、密钥）
- 修改系统级配置

### 🎯 特殊规则

#### 自动降级到 Level 1 的文件类型
- 测试文件: `**/*test*.py`, `**/*spec*.py`
- 文档文件: `**/*.md`, `**/*.txt`
- 临时文件: `**/__pycache__/**`, `**/*.pyc`

#### 快速切换指令

用户可随时使用以下指令切换授权级别：

- "切换到 Level 1" 或 "完全自动模式"
- "切换到 Level 2" 或 "显示模式"（默认）
- "切换到 Level 3" 或 "确认模式"
- "切换到 Level 4" 或 "严格模式"
- "切换到 Level 5" 或 "禁止模式"
- "当前授权级别" → 显示当前配置
